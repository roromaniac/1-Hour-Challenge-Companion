# 1-Hour-Challenge-Companion
The official modpack for use with Kingdom Hearts II Final Mix Randomizer's 1 Hour Challenge Category.

This modpack must be placed ABOVE ALL OTHER MODS, INCLUDING THE SEED. The modpack includes the following:
- Crazycatz00's Moose RAM expansion
- 123456789num's Better Boss Scaling
- DA's menu before Dark Thorn
- Sonicshadowsilver2's Remove Swarm
- JaredWeakstrike's 100 Acre Orb
- JaredWeakstrike's Engine Core Skip
- Geek's Bonus Speed Up
